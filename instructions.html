Instructions to Create Puzzle

Following are the steps to create a puzzle:

  1. Create a puzzle where the solution is an alphanumeric string (solely numerical is preferred).
  2. Write up the puzzle in plain text. If you would like to use images, please use an external service to store the image. Please ensure your puzzle is explained clearly enough that it can be solved based on just the information given in the text.
  3. Choose a title and reward (number of AR) for your puzzle. The reward will immediately be taken from your wallet when you create the puzzle, and it will only be available to whoever enters the solution into the puzzle page.
  4. Head to the Create Puzzle Page and enter your puzzle information to create the puzzle and post it to the blockchain.
